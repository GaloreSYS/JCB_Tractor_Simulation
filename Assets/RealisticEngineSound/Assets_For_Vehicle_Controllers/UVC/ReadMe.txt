If you want to use Realistic Engine Sounds 2 with Universal Vehicle Controller,
import Universal Vehicle Controller (make sure you done all required steps to make UVC work), then import Realistic Engine Sounds 2 and later UVC_RES2-Lite.unitypackage.
This unitypackage contain demo scenes, scripts for demo scenes and a prefab for Universal Vehicle Controller.
If you don't own Universal Vehicle Controller, you can buy it here: https://assetstore.unity.com/packages/templates/systems/universal-vehicle-controller-176314?aid=1101l3r5B
