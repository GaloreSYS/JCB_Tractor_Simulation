If you want to use Realistic Engine Sounds 2 with Realistic Traffic Controller,
import Realistic Traffic Controller, Realistic Engine Sounds 2 and later RTC_RES2-Lite.unitypackage.
This unitypackage contain demo scenes, a script and a prefab made for Realistic Traffic Controller compatibility.
If you don't own Realistic Traffic Controller, you can buy it here: https://assetstore.unity.com/packages/tools/behavior-ai/realistic-traffic-controller-258961?aid=1101l3r5B

Read the documentation for tutorials.
