If you want to use Realistic Engine Sounds 2 with BoneCracker Games' Realistic Car Controller,
import Realistic Car Controller, Realistic Engine Sounds 2 and later RCC_RES2-Lite.unitypackage.
This unitypackage contain demo scenes, a script and a prefab made for Realistic Car Controller compatibility.
If you don't own Realistic Car Controller, you can buy it here: https://assetstore.unity.com/packages/tools/physics/realistic-car-controller-16296?aid=1101l3r5B

Read the documentation for tutorials.
