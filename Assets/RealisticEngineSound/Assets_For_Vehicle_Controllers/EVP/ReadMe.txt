If you want to use Realistic Engine Sounds 2 with Edy's Vehicle Physics,
import Edy's Vehicle Physics, Realistic Engine Sounds 2 and later EVP_RES2-Lite.unitypackage.
This unitypackage contain demo scenes, a script and a prefab made for Edy's Vehicle Physics compatibility.
If you don't own Edy's Vehicle Physics, you can buy it here: https://assetstore.unity.com/packages/tools/edy-s-vehicle-physics-403?aid=1101l3r5B

Read the documentation for tutorials.
