If you want to use Realistic Engine Sounds 2 with Vehicle Physics Pro,
import Vehicle Physics Pro, Realistic Engine Sounds 2 and VPP_RES2-Lite.unitypackage.
This unitypackage contain demo scenes, a script for demo scenes and a prefab for Vehicle Physics Pro compatibility.
Get Vehicle Physics Pro here: https://assetstore.unity.com/packages/tools/physics/vehicle-physics-pro-community-edition-153556?aid=1101l3r5B

Read the documentation for tutorials.
