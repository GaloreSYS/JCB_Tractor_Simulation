If you want to use Realistic Engine Sounds 2 with NWH2 Vehicle Physics,
import NWH2 Vehicle Physics, Realistic Engine Sounds 2 and NWH2_RES2-Lite.unitypackage into your project.
This unitypackage contain demo scenes, scripts for demo scenes and a prefab for NWH2 Vehicle Physics compatibility.

Get NWH2 Vehicle Physics here: https://assetstore.unity.com/packages/tools/physics/nwh-vehicle-physics-2-166252?aid=1101l3r5B

Read the documentation for tutorials.
